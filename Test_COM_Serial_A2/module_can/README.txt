
CAN Bus Module
**************

scope:
   General Use

description:
   CAN Bus component

keywords:
   CAN,Rx,Tx

boards:
   XA-SK-ISBUS


Features
========

   * frame id filtering,

   * configurable size rx frame buffer,

   * Fully CAN 2.0A and 2.0B compilant.

   * Up to 1Mb baud rate

   * Support for extended identifier(29-bit) for PHY layer
