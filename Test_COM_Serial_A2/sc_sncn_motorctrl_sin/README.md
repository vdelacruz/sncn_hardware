Key Features
---------
   * Sinusoidal Commutation
   * Profile Position Control 
   * Profile Velocity Control
   * Profile Torque Control
   * Ethercat Operating Modes
   * Support QEI sensor with Index/ no Index
   * Support Hall sensor
   * Support external Analog sensor
